没有理由将变量重新赋值给它自己。要么这个语句是冗余的，应该删除，要么此赋值语句错误，而其他一些值或变量本来是用于赋值的。

不合规的代码示例

func (user *User) rename(name string) {
  name = name  // Noncompliant
}

合规的解决方案

func (user *User) rename(name string) {
  user.name = name
}

请参阅

  • CERT, MSC12-C. - Detect and remove code that has no effect or is never executed
