嵌套的 when 结构很难理解，因为你很容易将内部 when 的情况混淆为属于外部语句。 因此，应避免使用嵌套的 when 语句。

具体来说，您应该构造代码以避免需要嵌套的 when 语句，但如果不能，则考虑将内部 when 移动到另一个函数。

不合规的代码示例

fun foo(n: Int, m: Int) {
  when (n) {
    0 ->
      when (m) {  // 不合规； 嵌套的 when
        // ...
      }
    1 -> print("1")
    else -> print("2")
  }
}

合规的解决方案

fun foo(n: Int, m: Int) {
  when (n) {
    0 -> bar(m)
    1 -> print("1")
    else -> print("2")
  }
}

fun bar(m: Int){
  when(m) {
    // ...
  }
}
