将具有相同实现的所有分支置于 switch 或 if 链中是错误的。要么发生了复制粘贴错误，要么应该执行不同的操作，或者根本不应该有 switch/if 链。

不合规的代码示例

if b == 0 {  // 不合规
  doOneMoreThing()
} else {
  doOneMoreThing()
}

switch i {  // 不合规
  case 1:
    doSomething()
  case 2:
    doSomething()
  case 3:
    doSomething()
  default:
    doSomething()
}

例外

此规则不适用于 if 链没有 else， or 或 switch 没有 default 子句。

if b == 0 {    //没问题，这可能是为了使代码更具可读性
  doSomething()
} else if b == 1 {
  doSomething()
}
