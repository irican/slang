当两种方法具有相同的实现时，要么是错误的——有其他意图——或者是故意重复的，但这可能会让维护者感到困惑。 在后一种情况下，一个实现应该调用另一个。 不考虑数字和字符串文字。

不合规的代码示例

class Box(length: Int, width: Int, height: Int) {
  def volume: Int = {
    val s = length * width
    s * height
  }

  def area: Int = {
    val s = length * width
    s * height
  }
}

合规的代码示例

class Box(length: Int, width: Int, height: Int) {
  def volume: Int = {
    val s = length * width
    s * height
  }

  def area: Int = {
    length * width
  }
}

例外

忽略少于2个语句的方法。
