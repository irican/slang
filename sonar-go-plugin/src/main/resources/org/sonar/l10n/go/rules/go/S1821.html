嵌套的 switch 结构很难理解，因为您很容易将内部 switch 的case混淆为属于外部的语句。 因此，应避免使用嵌套的 switch 语句。

具体来说，您应该构造代码，以避免嵌套 switch 语句，但是如果不能，则考虑将内部 switch 转移到另一个函数。

不合规的代码示例

func foo(x,y int) {
	switch x {
	case 0:
		switch y { // Noncompliant; nested switch
		// ...
		}
	case 1:
		// ...
	default:
		// ...
	}
}

合规的解决方案

func foo(x,y int) {
	switch x {
	case 0:
		bar(y)
	case 1:
		// ...
	default:
		// ...
	}
}

func bar(y int) {
	switch y {
	// ...
	}
}
