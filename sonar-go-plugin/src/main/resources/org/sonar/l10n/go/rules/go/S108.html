大多数情况下，当代码块是空的时，一段代码真的丢失了。 因此必须填充或移除这样的空块。

不合规的代码示例

func compute(a int, b int) {
	sum := a + b
	if  sum > 0 { } // 不合规； 是故意为空还是缺少一段代码？
	fmt.Println("Result:", sum)
}

合规的解决方案

func compute(a int, b int) {
	sum := a + b
	if  sum > 0 {
		fmt.Println("Positive result")
	}
	fmt.Println("Result:", sum)
}

例外

当代码块包含注释时，此块不被视为空。 带有空代码块的 for 和 select 语句也被忽略。
