将具有相同实现的所有分支置于 case 或 if 链中是错误的。 要么发生了复制粘贴错误，要么应该执行不同的操作，或者根本不应该有 case/if 。

不合规的代码示例

if b == 0  # 不合规
  doOneMoreThing()
else
  doOneMoreThing()
end

b = a > 12 ? 4 : 4;  # 不合规

case i  # 不合规
  when 1
    doSomething()
  when 2
    doSomething()
  when 3
    doSomething()
  else
    doSomething()
end

例外

此规则不适用于 if 链或 case 没有 else的情况。

if b == 0 # 没问题，这可能是为了使代码更具可读性
  doSomething()
elsif b == 1
  doSomething()
end
