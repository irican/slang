最后的 default 子句的目的是防御性编程。该子句应采取适当行为，或包含适当的注释，说明为何不采取任何行为。

不合规的代码示例

switch tag { // 不合规——缺少default子句
case 0, 1, 2, 3:
	foo()
case 4, 5, 6, 7:
	bar()
}

合规的解决方案

switch tag {
case 0, 1, 2, 3:
	foo()
case 4, 5, 6, 7:
	bar()
default:
	qix()
}
