嵌套的 case 结构很难理解，因为你很容易将内部 case 的情况混淆为属于外部语句。 因此，应避免使用嵌套的 case 语句。

具体来说，您应该构造代码以避免需要嵌套的 case 语句，但如果不能，则考虑将内部 case 移动到另一个函数。

不合规的代码示例

def foo(n, m)
  case n
  when 0
      case m  # 不合规； 嵌套的 case
      when 0 then puts "0"
        # ...
      end
  when 1 then puts "1"
  else puts "2"
  end
end

合规的解决方案

def foo(n, m)
  case n
  when 0
    bar(m)
  when 1 then puts "1"
  else puts "2"
  end
end

def bar(m)
  case m
  when 0 then puts "0"
    # ...
  end
end
