跳转语句（ return ， break 和 next ）将控制流移出当前代码块。 因此，跳转后的任何语句都是死代码。

不合规的代码示例

def foo(a)
  i = 10
  return a + i    # 不合规
  i += 1          # 死代码
end

合规的解决方案

def foo(a)
  i = 10
  return a + i
end
