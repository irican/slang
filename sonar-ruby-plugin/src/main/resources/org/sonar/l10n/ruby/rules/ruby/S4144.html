当两种方法具有相同的实现时，要么是错误的——有其他意图——或者是故意重复的，但这可能会让维护者感到困惑。 在后一种情况下，一个实现应该调用另一个。 不考虑数字和字符串文字。

不合规的代码示例

class Box
  def initialize(length, width, height)
    @length, @width, @height = length, width, height
  end

  def getVolume
    area = @length * @width
    return area * @height
  end

  def getArea                  # 不合规， 实现与 “getVolume”相同
    area = @length * @width
    return area * @height
  end
end

合规的代码示例

class Box
  def initialize(length, width, height)
    @length, @width, @height = length, width, height
  end

  def getVolume
    return getArea * @height
  end

  def getArea
    return @length * @width
  end
end

例外

忽略少于2个语句的方法。
