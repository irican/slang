将具有相同实现的所有分支置于 when 或 if 链中是错误的。 要么发生了复制粘贴错误，要么应该执行不同的操作，或者根本不应该有 when/if 链。

不合规的代码示例

if (b == 0) {  // 不合规
     doOneMoreThing()
} else {
     doOneMoreThing()
}

when (i) {  // 不合规
    1 -> doSomething()
    2 -> doSomething()
    3 -> doSomething()
    else -> doSomething()
}

例外

此规则不适用于 if 链没有 else， 或 when 没有 else 子句的情况。

if (b == 0) {
    doOneMoreThing()
} else if (b == 1) {
    doOneMoreThing()
}
