大多数情况下，当一段代码缺失时，代码块是空的。因此必须填充或移除这样的空块。

不合规的代码示例

def compute(a, b)
  sum = a + b
  if  sum > 0 # 不合规；是故意为空还是缺少一段代码？
  end
  puts "Result: #{sum}"
end

合规的解决方案

def compute(a, b)
  sum = a + b
  if  sum > 0
    puts "Positive result"
  end
  puts "Result: #{sum}"
end

例外

当代码块包含注释时，此块不被视为空。

while 与 unless 循环也是这一规则的例外。

while @order.process_next; end # 合规
