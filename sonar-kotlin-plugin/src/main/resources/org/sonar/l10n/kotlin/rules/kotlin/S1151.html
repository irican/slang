when 语句只应用于在控制流中明确定义一些新分支。只要一个 case 子句包含太多语句，就会大大降低整个控制流语句的可读性。 在这种情况下，应将case子句的内容提取到专用方法中。

不合规的代码示例

默认阈值为5：

when (myVariable) {
  0 -> {// 不合规: 到下一个case之间有6行
    methodCall1("");
    methodCall2("");
    methodCall3("");
    methodCall4("");
    }
  1 -> {
    ...
  }
}

合规的解决方案

when (myVariable) {
  0 -> doSomething()
  1 -> {
    ...
  }
}
...
fun doSomething() {
    methodCall1("");
    methodCall2("");
    methodCall3("");
    methodCall4("");
}
